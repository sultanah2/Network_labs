# Network_labs
Overview

This repository contains networking laboratory exercises completed during my university coursework using Cisco Packet Tracer.

The labs demonstrate practical experience with routing, switching, VLAN implementation, and network security fundamentals. Each lab includes the Packet Tracer project file and a network topology diagram.
